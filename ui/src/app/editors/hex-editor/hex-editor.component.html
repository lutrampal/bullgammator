L’éditeur de code va charger dans le Gamma ET le code à exécuter.
Un premier onglet permet l’édition du tableau de connexions (série 3) et un deuxième l’édition du tambour. Lors de la validation (bouton "Charger le code"), le texte source est analysé : les commentaires, les espaces, les retours et tabulations sont retirés. Puis, s’il n’y a aucune erreur, le code est écrit dans la mémoire correspondante ("série 3" ou "tambour").

Le jeu d'instructions s'avère être indispensable pour programmer le Gamma 3. De la documentation complémentaire est disponible ici, notamment le polycopié du cours de Louis Bolliet.
